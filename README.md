# 🧠 Speech-Based Parkinson’s Disease Detection

This project focuses on detecting Parkinson’s disease using speech signal processing techniques. It compares **time domain features** and **frequency domain features** to evaluate their effectiveness using machine learning.

---

## 📌 Objective

To analyze and compare:

- **Time Domain Features**: STE, ZCR, Pitch, Jitter, Shimmer  
- **Frequency Domain Features**: MFCC, Delta MFCC, Spectral Features, Chroma  

for detecting Parkinson’s disease from speech signals.

---

## ⚙️ Technologies Used

- Python  
- Librosa (Audio Processing)  
- NumPy  
- Scikit-learn (SVM Model)  
- Matplotlib
